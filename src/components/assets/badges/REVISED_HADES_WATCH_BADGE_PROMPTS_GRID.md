# Revised Hades Watch Mission Badge Prompts

These prompts are rebuilt for **Nano Banana 2** with stricter badge production requirements.

## Core Art Rules

Every badge should follow these rules:

- square badge
- rounded corners
- badge art fills the full square
- no outside background
- any area outside the rounded square must be transparent
- no sticker peel corners
- no placeholder text
- no extra slogans
- the **full badge name** must appear on the badge
- no other text unless it is subtle in-world code or decorative micro-labeling integrated into the badge design
- consistent badge family style across the whole set
- readable at website badge size
- clean silhouette
- no character portraits
- no weapons
- no gore
- no photorealism
- no corporate logo style

---
