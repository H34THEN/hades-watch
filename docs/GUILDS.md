# Guilds // Underwatch Cells

Player-created community groups (not factions).

## Flow

1. Approved user submits guild request at `/community/guilds`
2. Status: `PENDING_REVIEW`
3. Moderator approves → `APPROVED`, founder becomes leader
4. Other users request membership

## Rules

- Cannot impersonate official factions
- Nonviolent conduct agreement required
- Admin can reject, suspend, archive

## Badge hooks

- `guild-founder`, `guild-steward`, `circle-keeper`
