# Forum Signatures

Signatures appear below forum posts when not hidden by user preference.

## Modes

- **Compact** — 88×31 button + short text line
- **Full** — button + larger banner + text
- **Hidden** — no signature rendered

## Creation modes

1. **Generated** — `ForumSignatureBuilder` stores safe JSON in `ForumSignatureAsset.safeConfig`, rendered by `NetNeighborButton`
2. **Uploaded** — PNG/JPG/WEBP only, max 2 MB, stored in `storage/uploads/forum-signatures/`

## Presets

Button sizes: 88×31, 120×40, 180×60, 234×60  
Banner sizes: above + 468×60, 96×96  

Themes: Dead Index Violet, Styx Neon Rat, Asclepian Green Pulse, Oracular Signal Blue, and others from the shared banner builder gradient presets.

## Safety note (shown in UI)

Forum signatures use safe tokens and trusted components. Raw scripts, iframes, tracking pixels, and unsafe CSS are not allowed.
