# Hades Watch Theme System

Initial planned themes:

- Ghost in the Machine
- Terminal Kiddie
- Fallout CRT
- Neon Oracle
- Corporate Dystopia
- Underworld Ops
- Null Signal
- Arcade Necropolis

Themes should use CSS variables and eventually support asset slots for:

- logo
- favicon
- backgrounds
- texture overlays
- cursor images
- sounds
- ambience
