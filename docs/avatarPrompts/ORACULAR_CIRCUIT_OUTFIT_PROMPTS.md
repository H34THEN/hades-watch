# Oracular Circuit Outfit Set

## Avatar Forge Scope

This Markdown file contains the full modular avatar outfit prompt spread for **The Oracular Circuit** specialty avatar item set.

This set is designed for use with **Hades Watch Avatar Forge** and the approved Hades Watch masculine and feminine base bodies.

The spread includes:

1. Masculine top
2. Feminine top
3. Masculine pants
4. Feminine pants
5. Masculine jacket
6. Feminine jacket
7. Signal visor
8. Cipher gloves
9. Ghost packet satchel

These prompts are built for modular avatar-builder layers.

## Visual Direction

The Oracular Circuit set should feel:

- functional
- signal-literate
- sleek but not corporate
- techno-mystic
- underworld systems-worker cool
- ghost-packet ready
- cipher operative fieldwear
- readable at avatar scale

Use:

- black glass
- graphite
- smoke gray
- matte black
- electric blue
- deep violet
- signal cyan
- bone white notation marks
- brushed silver hardware
- restrained fluorescent accents

Do not use:

- real hacker logos
- malware symbols
- real-world code branding
- military goggles
- police styling
- cyber-ninja armor
- giant glowing VR helmets
- flashy neon overload
- readable private data
- straight-on accessories

## Prompt 1: Oracular Circuit Masculine Top

```txt
Create a modular upper-body clothing layer for the Hades Watch avatar builder: Oracular Circuit masculine signal-reader top, designed specifically to fit the approved masculine Hades Watch base body exactly.

This is a pose-alignment task first and a clothing-design task second.

Use the approved masculine base body image as the exact placement reference. The top must be generated as if it is painted directly over the approved masculine base body without moving, rotating, straightening, mirroring, reposing, or changing the body in any way.

CRITICAL POSE PRESERVATION:
Do not change the pose. Do not redraw the body. Do not correct the stance. Do not straighten the character. Do not make the figure front-facing. Do not mirror the body. Do not flip the garment horizontally. Do not reverse the 3/4 angle. Do not create a new clothing model pose. Do not change the body angle, torso angle, shoulder line, arm placement, hand placement, body scale, canvas margins, or facing direction.

FACING DIRECTION LOCK:
The top must face the exact same direction as the approved masculine base body. Do not reverse which shoulder appears forward. Do not reverse the side seam perspective. Do not make the garment symmetrical or straight-on. The garment must lay naturally over the existing masculine base body’s angled form.

Generate only the top layer. Do not include the body, skin, face, hair, hands, pants, jacket, shoes, props, or background. The output should be a transparent clothing layer that fits directly on top of the approved masculine base body.

Design:
Create a masculine-cut Oracular Circuit signal-reader top that feels like underworld cipher operative clothing. It should look practical, technical, and slightly ritualistic, like a base layer worn by someone who reads machine prophecy and teaches it to lie. Use matte black, graphite, smoke gray, and black-glass panel accents, with restrained electric-blue and violet signal piping, tiny signal-cyan stitch marks, bone-white notation ticks, and brushed silver fasteners. The silhouette should be fitted but not tight, clean, layer-friendly, and compatible with a jacket.

Add Oracular Circuit design language:
- subtle circuit-trace seam embroidery
- ghost-packet tab details
- encrypted veil panel references
- small oracle-eye inspired glyph marks
- black-glass fastening tabs
- raised collar or compact structured neckline
- long fitted sleeves
- clean wrist seam detailing
- silver hardware instead of gold
- fluorescent accents used as restrained signal markers only

Fit notes:
Make the top specifically compatible with the approved masculine base body’s neck, shoulders, chest, torso, waist, and upper arms. It should complement a broader shoulder line and straighter torso without becoming armor, tactical gear, or a robe.

Style:
Hades Watch cyberpunk gothic avatar-builder style, modernized early 2000s Gaia Online charm, polished semi-stylized realism, clean line art, soft cel shading, readable silhouette, practical modular game-ready asset design, technomythic underworld resistance aesthetic.
```
