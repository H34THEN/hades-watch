## Avatar Forge Scope

This Markdown file contains a safe modular avatar accessory prompt spread for **Hades Watch special unlock tech gear**.

These prompts are designed for use with **Hades Watch Avatar Forge** and the approved Hades Watch masculine and feminine base bodies.

These are fictional avatar customization assets only. They are not real-world uniforms, tactical equipment, hacking tools, surveillance devices, medical devices, weapons, operational gear, or instructions.

The spread includes:

1. Memory Moth Ear Cuff

5. Ghostlight Pocket Projector

9. Charon Transit Card Holder
10. Sunseed Tech Pendant

These prompts are built for modular avatar-builder layers.

## Visual Direction

The special unlock tech accessories should feel:

- playful cyberpunk trinkets
- underworld archive tech
- fictional signal charms
- wearable avatar collectibles
- small, clever, and non-threatening
- gothic but cute
- repaired, personalized, and game-ready
- myth-tech without real-world functionality
- premium unlocks without looking royal or corporate

Use:

- matte black
- charcoal
- smoke gray
- bone white
- pomegranate red
- brushed silver
- black glass
- signal cyan
- soft violet
- muted amber
- restrained fluorescent accents

Do not use:

- weapons
- real logos
- brand logos
- official insignia
- police or military styling
- tactical styling
- real hacker symbols
- malware symbols
- readable private data
- medical symbols
- red cross symbols
- hate symbols
- real gang symbols
- political slogans
- property-damage instructions
- straight-on product renders unless specifically requested

## Prompt 1: Memory Moth Ear Cuff

```txt
Create a transparent modular accessory overlay for the Hades Watch avatar builder, using the approved masculine and feminine base body image as the locked template. This is an overlay extraction task first and an accessory-design task second.

Use the approved base body as the exact placement reference. First fit the ear cuff directly onto the approved base body’s visible ear area, head angle, jawline angle, neck angle, pose, and screen-facing direction. Then output only the transparent ear cuff layer in the same canvas position.

The accessory must inherit the base body’s exact screen-facing direction, 3/4 angle, head angle, visible side, ear placement, jawline relationship, canvas placement, and scale. Do not generate a standalone jewelry item. Do not mirror, flip, rotate, straighten, or reinterpret the accessory.

CRITICAL ACCESSORY ALIGNMENT:
This accessory must lay over the approved base body in the same 3/4 perspective. Do not create a straight-on product image. Do not reverse the facing direction. Do not change the head, neck, shoulder, or body angle. The accessory must sit naturally on the approved base body as a transparent modular avatar layer.

Placement:
The ear cuff must align to the visible ear or visible side of the head. It should remain small and wearable with different hairstyles.

Design:
Create a Memory Moth Ear Cuff for a fictional avatar character. It should look like a small black-glass and brushed-silver moth-shaped tech charm with bone-white wing etching, pomegranate-red micro dots, and tiny signal-cyan glow accents. It should feel like a symbolic archive trinket, not a real device. Keep it delicate, gothic, readable at avatar scale, and non-threatening.

Style:
Hades Watch cyberpunk gothic avatar-builder style, polished semi-stylized realism, clean line art, soft cel shading, readable silhouette, modular game-ready accessory design.

Canvas:
1024x1024 square canvas, transparent background. Only the ear cuff layer, positioned exactly where it would sit on the approved base body.

Negative prompt:
full character, body, face, hair, background, changed pose, straight-on product image, wrong head angle, wrong facing direction, mirrored accessory, oversized jewelry, official insignia, weapon-like shape, real logo, readable text, private data, watermark, photorealism, AI artifacting, unusable layer edges
```
