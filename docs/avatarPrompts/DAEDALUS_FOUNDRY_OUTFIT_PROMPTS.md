# Daedalus Foundry Outfit Set

## Avatar Forge Scope

This Markdown file contains the full modular avatar outfit prompt spread for **The Daedalus Foundry** specialty avatar item set.

This set is designed for use with **Hades Watch Avatar Forge** and the approved Hades Watch masculine and feminine base bodies.

The spread includes:

1. Masculine top
2. Feminine top
3. Masculine pants
4. Feminine pants
5. Masculine jacket
6. Feminine jacket
7. Labyrinth tool belt
8. Schematic wrist slate
9. Repair tag bandolier

These prompts are built for modular avatar-builder layers.

## Visual Direction

The Daedalus Foundry set should feel:

- repair culture
- maker infrastructure
- hardware hacker utility
- toolsmith fieldwear
- workshop practical
- liberatory engineering
- anti-surveillance craft
- build the exit, hide the door
- non-weaponized

Use:

- gunmetal
- workshop black
- warm gray
- brass
- bronze
- brushed silver
- blueprint blue
- forge amber
- bone-white schematic marks
- restrained fluorescent worklight accents

Do not use:

- weaponsmith aesthetics
- combat armor
- mecha armor
- military or police styling
- unsafe tool imagery
- corporate industrial logos
- giant flashy neon
- straight-on accessories

## Prompt 1: Daedalus Foundry Masculine Top

```txt
Create a modular upper-body clothing layer for the Hades Watch avatar builder, designed specifically to fit the approved masculine Hades Watch base body exactly. This is a pose-alignment task first and a clothing-design task second.

Use the approved masculine base body image as the exact placement reference. The layer must be generated as if it is painted directly over the approved masculine base body without moving, rotating, straightening, mirroring, reposing, or changing the body in any way.

CRITICAL DIRECTION LOCK:
The item must face the exact same screen direction as the approved base body. Do not mirror it. Do not flip it. Do not reverse the 3/4 angle. Preserve the same screen-side shoulder, torso side, hip side, arm, leg, and seam perspective as the approved masculine base body.

CRITICAL POSE PRESERVATION:
Do not change the pose. Do not redraw the body. Do not straighten the stance. Do not make the figure front-facing. Do not create a new model pose. Preserve the exact body angle, facing direction, canvas placement, scale, shoulder angle, torso angle, hip angle, arm placement, hand placement, leg placement, knee placement, ankle placement, and screen-side forward limb relationship.

Generate only the upper-body clothing layer. Do not include the body, skin, face, hair, hands, unrelated clothing, shoes, props, or background. Use a transparent background.

Design:
Create a masculine-cut Daedalus Foundry maker top that feels like practical workshop fieldwear for an engineer, toolsmith, repair worker, or hardware hacker. It should look sturdy, clever, and functional without becoming armor or corporate workwear. Use workshop black, gunmetal, warm gray, bone-white schematic stitch marks, blueprint-blue seam details, forge-amber worklight accents, brushed silver fasteners, and restrained brass or bronze hardware.

Add Daedalus Foundry design language:
- reinforced work seams
- schematic-line stitching
- subtle labyrinth engineering marks
- compact utility panels
- practical raised collar or crew neck
- fitted but not tight sleeves
- small repair-tag tabs
- non-weaponized maker identity
- no combat armor
- no corporate branding

Fit notes:
Make the upper-body clothing specifically compatible with the approved masculine base body’s neck, shoulders, chest, torso, waist, and upper arms. Preserve exact pose, direction, scale, layer fit, and 3/4 perspective.

Style:
Hades Watch cyberpunk gothic avatar-builder style, modernized early 2000s Gaia Online charm, polished semi-stylized realism, clean line art, soft cel shading, readable silhouette, practical modular game-ready asset design, technomythic underworld resistance aesthetic.

Canvas:
1024x1024 square canvas, transparent background. Only the requested layer, positioned exactly where it would sit on the approved base body.

Negative prompt:
```
