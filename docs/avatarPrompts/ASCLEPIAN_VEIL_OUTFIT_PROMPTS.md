# Asclepian Veil Outfit Set

## Avatar Forge Scope

This Markdown file contains the full modular avatar outfit prompt spread for **The Asclepian Veil** specialty avatar item set.

This set is designed for use with **Hades Watch Avatar Forge** and the approved Hades Watch masculine and feminine base bodies.

The spread includes:

1. Masculine top
2. Feminine top
3. Masculine pants
4. Feminine pants
5. Masculine jacket
6. Feminine jacket
7. Field stabilizer satchel
8. Triage wrist cuff
9. Privacy veil scarf

These prompts are built for modular avatar-builder layers.

## Visual Direction

The Asclepian Veil set should feel:

- calm under pressure
- clandestine underworld clinicwear
- field stabilizer gear
- harm-reduction worker utility
- medical privacy defense
- soft but unbreakable
- care as infrastructure
- survival as sabotage
- non-militarized

Use:

- worn black
- charcoal
- muted gray
- bone white
- faded clinic green
- desaturated teal
- muted crimson
- brushed silver hardware
- soft fluorescent clinic-signal accents used sparingly

Do not use:

- red cross symbols
- real medical logos
- modern hospital branding
- military medic gear
- police or tactical styling
- combat armor
- clinical advice imagery
- flashy neon overload
- straight-on accessories
