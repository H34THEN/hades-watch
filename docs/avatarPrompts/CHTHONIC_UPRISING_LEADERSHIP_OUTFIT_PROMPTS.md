# Chthonic Uprising Leadership Outfit Set

## Avatar Forge Scope

This Markdown file contains the full modular avatar outfit prompt spread for the **Chthonic Uprising Leadership** specialty avatar item set.

This set is designed for use with **Hades Watch Avatar Forge** and the approved Hades Watch masculine and feminine base bodies.

The spread includes:

1. Masculine top
2. Feminine top
3. Masculine pants
4. Feminine pants
5. Masculine jacket
6. Feminine jacket
7. Dead Index fanny pack
8. Archivist digital techno staff
9. Underwatch signal cuff

These prompts are built for modular avatar-builder layers.

## Visual Direction

The Chthonic Uprising Leadership set should feel:

- functional
- techno-punk cool
- specialty-tier without being flashy
- patched and repaired
- practical underworld leadership gear
- archive-punk
- field-ready
- dangerous to tyrants, safe to erased people

Use:

- matte black
- charcoal
- smoke gray
- bone white
- pomegranate red
- brushed silver
- gunmetal silver
- restrained fluorescent signal accents

Do not use:

- gold
- royal styling
- military styling
- police styling
- flashy neon fashion
- giant glowing details
- real logos
- readable branded text

## Prompt 1: Chthonic Uprising Masculine Top

```txt
Create a modular upper-body clothing layer for the Hades Watch avatar builder: Chthonic Uprising masculine field top, designed specifically to fit the approved masculine Hades Watch base body exactly.

This is a pose-alignment task first and a clothing-design task second.

Use the approved masculine base body image as the exact placement reference. The top must be generated as if it is painted directly over the approved masculine base body without moving, rotating, straightening, mirroring, reposing, or changing the body in any way.

CRITICAL POSE PRESERVATION:
Do not change the pose. Do not redraw the body. Do not correct the stance. Do not straighten the character. Do not make the figure front-facing. Do not mirror the body. Do not flip the garment horizontally. Do not reverse the 3/4 angle. Do not create a new clothing model pose. Do not change the body angle, torso angle, shoulder line, arm placement, hand placement, body scale, canvas margins, or facing direction.

FACING DIRECTION LOCK:
The top must face the exact same direction as the approved masculine base body. Do not reverse which shoulder appears forward. Do not reverse the side seam perspective. Do not make the garment symmetrical or straight-on. The garment must lay naturally over the existing masculine base body’s angled form.

Generate only the top layer. Do not include the body, skin, face, hair, hands, pants, jacket, shoes, props, or background. The output should be a transparent clothing layer that fits directly on top of the approved masculine base body.

Design:
Create a Chthonic Uprising masculine field top that looks like practical underworld resistance wear. It should feel like a cross between a fitted archive-worker shirt, a protest-organizer underlayer, and a technopunk signal-operative base layer. Use matte black, charcoal, and smoke gray as the core colors, with bone-white stitch lines, pomegranate-red patch accents, brushed silver fasteners, and very restrained fluorescent signal piping in small amounts. The silhouette should be fitted but not tight, practical, layer-friendly, and clearly wearable under a vest or jacket.

Add Chthonic Uprising design language:
- patched repair seams
- redaction-strip panel details
- subtle Dead Index glyph marks
- archive-tab stitching
- punk utility seam structure
- a practical neckline or short raised collar
- long fitted sleeves
- a clean resistance silhouette
- silver hardware instead of gold
- very small fluorescent accent details used as signal stitching, not decorative neon fashion

Fit notes:
Make the top specifically compatible with the approved masculine base body’s neck, shoulders, chest, torso, waist, and upper arms. It should complement a broader shoulder line and straighter torso while remaining grounded and practical. Do not make it armor, tactical gear, or ceremonial clothing.

Style:
Hades Watch cyberpunk gothic avatar-builder style, modernized early 2000s Gaia Online charm, polished semi-stylized realism, clean line art, soft cel shading, readable silhouette, practical modular game-ready asset design, technomythic underworld resistance aesthetic.

Canvas:
1024x1024 square canvas, transparent background. Only the top layer, positioned exactly where it would sit on the approved masculine base body.
```
