# Myrmidon Grinders Outfit Set

## Avatar Forge Scope

This Markdown file contains the full modular avatar outfit prompt spread for **The Myrmidon Grinders** specialty avatar item set.

This set is designed for use with **Hades Watch Avatar Forge** and the approved Hades Watch masculine and feminine base bodies.

The spread includes:

1. Masculine top
2. Feminine top
3. Masculine pants
4. Feminine pants
5. Masculine jacket
6. Feminine jacket
7. Gatewatch belt pouch
8. Contact tree armband
9. Route marker scarf

These prompts are built for modular avatar-builder layers.

## Visual Direction

The Myrmidon Grinders set should feel:

- community defense
- logistics crew utility
- protest marshal influence without militarization
- strength without cruelty
- gatewatch steward gear
- sturdy, repaired, disciplined, and practical
- worker-built, not soldier-coded
- civic safety infrastructure

Use:

- matte black
- utility gray
- dark denim
- worn canvas
- amber warning marks
- rust accents
- bone white tabs
- brushed silver hardware
- restrained fluorescent amber safety accents

Do not use:

- weapons
- riot police styling
- military uniforms
- tactical combat armor
- intimidation aesthetics
- nationalist symbols
- police insignia
- real logos
- flashy neon overload
- straight-on accessories

Use matte black, utility gray, dark denim panels, worn canvas texture, bone-white tab stitches, rust accent stitching, brushed silver fasteners, and restrained fluorescent amber safety marks.

- reinforced workwear seams

- fitted but not tight sleeves
- reflective amber micro accents used sparingly

Make the upper-body clothing specifically compatible with the approved masculine base body’s neck, shoulders, chest, torso, waist, and upper arms. Preserve exact pose, direction, scale, layer fit, and 3/4 perspective.
