# Canon Tiers // Dead Index Authority

Six tiers for lore submissions at `/community/lore`:

| Tier | Label | Review |
|---|---|---|
| PRIVATE_DRAFT | Private Draft | Author only |
| CHARACTER | Character Canon | Light / auto path |
| GUILD | Guild Canon | Guild steward + mod |
| LOCAL_COMMUNITY | Local Community | Moderator |
| FACTION_ECHO | Faction Echo | Admin / lore steward |
| WORLD | World Canon | Owner / Archivist only |

Higher tiers never overwrite another user's character without consent or official canon without approval.

- `first-relic`, `guild-mythmaker`, `local-canon-scribe`, `lore-keeper`
