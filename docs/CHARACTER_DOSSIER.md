# Character Dossier

Route: `/profile/dossier`

## Purpose

Private operative record — structured identity source for the authenticated user. Not the decorative public Profile World.

## Shows

Callsign, display name, avatar preview, faction/alliance, account status, species, bio/tagline, visibility, badge/mission/cipher counts, reputation, edit links.

## Empty state

"No operative dossier has stabilized yet." → CTA **Stabilize Character** → `/mmo/character`

## Privacy

Private data stays here. Public presentation is `/profile/world/[callsign]`.
